## Mac卸载微信输入法

Mac上卸载微信输入法的详细步骤：

1. **退出微信输入法**
    - 点击屏幕右上角菜单栏的输入法图标
    - 选择"退出微信输入法"
2. **删除输入法配置**
    - 打开"系统设置"
    - 点击"键盘"
    - 选择"输入法"标签
    - 找到"微信输入法"并点击减号"-"将其移除
3. **删除应用程序**
    - 打开 Finder
    - 进入"应用程序"文件夹
    - 找到"微信输入法"
    - 将其拖入垃圾桶或右键选择"移到废纸篓"
4. **删除相关文件**
    - 打开 Finder
    - 按下 Command + Shift + G
    - 输入并访问以下路径：
        
        ```
        Code
        CopyInsert
        ~/Library/Input Methods/
        
        ```
        
    - 删除 **`WeChatInput.app`** 文件夹
    - 再次按下 Command + Shift + G
    - 输入并访问：
        
        ```
        Code
        CopyInsert
        ~/Library/Application Support/
        
        ```
        
    - 删除 **`WeChatInput`** 文件夹
5. **清空废纸篓**
    - 右键点击程序坞上的废纸篓
    - 选择"清倒废纸篓"

完成上面这些步骤后，微信输入法就被完全卸载了。如果之后想重新安装，可以从微信官网下载最新版本的输入法安装包。
