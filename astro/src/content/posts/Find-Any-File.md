# Find Any File：Mac 文件搜索找回

Spotlight 找不到你要的文件时，Find Any File 就是 Plan B。

**价格：付费** | **平台：macOS**

**官网：**[https://apps.tempel.org/FindAnyFile/](https://apps.tempel.org/FindAnyFile/)

---

## 核心功能

- 搜索 Spotlight 不索引的区域
- 按文件元数据精细搜索（日期、大小、类型等）
- 搜索隐藏文件和系统文件
- 搜索包内容
- 布尔搜索：AND/OR/NOT
