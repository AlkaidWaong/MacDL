# 功能介绍

### **一、隐私保护，守护你的上网安全**

Brave 浏览器以其强大的隐私保护功能而闻名，它通过以下方式保障您的上网安全：

- **Shields（护盾）**：内置强大的 Shields 功能，默认拦截广告、跟踪器、指纹识别、Cookies 等，保护您的隐私不被泄露。
- **Tor 模式**：提供 Tor 模式，通过 Tor 网络匿名浏览网页，隐藏您的 IP 地址和网络身份信息，实现真正的匿名浏览。
- **安全连接升级**：自动升级每个网页的 HTTPS 连接，防止中间人攻击，保障数据传输安全。

### **二、性能卓越，畅享极速浏览体验**

Brave 浏览器不仅注重隐私保护，还致力于为用户提供极致的浏览体验：

- **快速加载网页**：通过阻止广告和跟踪器等内容，减少页面加载负担，大幅提升网页加载速度。
- **资源高效利用**：设计为资源高效型浏览器，使用更少的 CPU 和内存，使设备运行更顺畅。

### **三、特色功能，满足你的多样化需求**

Brave 浏览器不仅是一款浏览器，更是一个功能强大的工具箱：

- **Brave 搜索**：独立的搜索引擎，不跟踪您的搜索行为，提供私密搜索体验。
- **Brave Rewards**：通过观看隐私保护的广告获得加密代币（BAT），支持内容创作者。
- **原生加密钱包**：集成安全的原生加密货币钱包，方便您管理加密资产。
- **Firewall + VPN**：保护您在每个设备上所有应用程序的网络连接安全。
- **Talk、News、Playlist**：提供视频通话、新闻订阅、播放列表等功能，满足您的多样化需求。
- **实用工具**：内置翻译、快速阅读、Wayback Machine 集成、IPFS 集成等实用工具，提升您的浏览效率。
