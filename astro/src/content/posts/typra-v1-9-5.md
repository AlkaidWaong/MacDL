# Typora：所见即所得的 Markdown 编辑器

Typora 的核心创新在于消灭了 Markdown 编辑器「左边写右边预览」的分裂感。

**价格：付费买断** | **平台：Mac + Windows + Linux**

**官网：**[https://typora.io/](https://typora.io/)

---

## 核心功能

- **所见即所得**：写出来就是最终样子
- **沉浸模式**：聚焦当前段落
- **主题系统**：多种主题可换
- **图片管理**：拖入图片自动管理
- **代码高亮**：支持多种语言
- **公式支持**：LaTeX 数学公式

---

## 亮点

- 写 Markdown 就像写 Word
- 专注、干净、不打扰
- 表格和代码块排版优秀

---

## 适合什么人

- 用 Markdown 写博客/文档的技术作者
- 不喜欢分屏预览的人
- 想要干净写作界面的人

---

## 总结

Typora 是 Markdown 编辑器的标杆产品之一。它把「写 Markdown」的门槛降到了几乎不用学的程度。
