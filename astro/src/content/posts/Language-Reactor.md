# Language Reactor概览

Language Reactor 是专为语言学习者设计的 Chrome 扩展程序。它支持多种语言，并提供了实时翻译功能，可以让用户在浏览网页时，通过双语对照的方式来学习新语言。

此外，该扩展程序还集成了语言学习辅助工具，如发音、单词笔记和例句等功能，帮助用户提高语言能力。

用户可以通过自定义设置来调整翻译的级别和样式，以适应个人的学习需求。Language Reactor 支持多种主流网站和服务，如 YouTube、Netflix 等，让学习语言变得更加轻松有趣。

## 功能

### 实时翻译

Language Reactor 提供实时翻译功能，帮助用户在观看视频时即时理解外语内容。无论是学习英语、法语还是其他语言，这项功能都能确保你跟上节奏，提升语言学习的效率与乐趣。

### 双语字幕

该扩展支持双语字幕，让用户一边看视频，一边对照翻译。通过这种互动式学习方式，用户能够更好地记忆单词和句子结构，快速提升语言能力。

根据用户的学习进度和偏好，Language Reactor 提供个性化的学习建议，使每个学习者的体验都独特而有效。这是确保用户持续进步的重要环节。

Language Reactor 能够与多个主流流媒体平台无缝对接，支持 Netflix、YouTube 等，令用户可以在他们最喜欢的平台上进行语言学习，轻松愉快。

除了视频内容，Language Reactor 还集成了发音、单词笔记和语境例句等丰富功能，全面提升用户的语言能力，满足不同层次学习者的需求。
