# PixPin for Mac：截图、标注、OCR 一步到位

PixPin 的特色不是「截什么」，而是「截下来还能做什么」。它在免费截图工具中提供了 OCR 识别和贴图这两个非常实用的功能。

**价格：免费** | **平台：Mac + Windows**

**下载：**[https://pixpin.cn/](https://pixpin.cn/)

---

## 核心功能详解

### OCR 文字识别
这是 PixPin 与 Xnip 最大的区别。截图后直接识别图中的文字并复制。适合：
- 从不可复制的文档/PDF 中提取文字
- 从网页截图里提取标题、数据
- 从 UI 设计稿里提取文案
- 从外国网站截图中提取文字再翻译

识别速度快，基本不需要等待。

### 标注工具
标注能力接近 CleanShot X 水平，包括箭头、高亮、矩形、椭圆、文字、马赛克、像素化等。

### 贴图（Pin images）
把截图固定在屏幕顶层做参考。支持同时固定多张。适合写文档时参考素材、做 UI 设计时对比效果。

### 长截图
滚动截取长网页、聊天记录。
